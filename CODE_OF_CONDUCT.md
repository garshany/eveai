# Code of Conduct

## Our Standard

This project should be practical, technical, and welcoming to contributors with different levels of EVE Online, TypeScript, and self-hosting experience.

Expected behavior:

- Be direct, respectful, and specific.
- Critique code and decisions, not people.
- Keep discussions focused on reproducible facts, tradeoffs, and project goals.
- Help maintain a safe open-source repository by avoiding secrets, private infrastructure details, and account data in public threads.

Unacceptable behavior:

- Harassment, threats, slurs, or personal attacks.
- Publishing private credentials, character/account secrets, private deployment details, or personal information.
- Repeated off-topic disruption after maintainers ask to stop.

## Enforcement

Maintainers may edit, hide, lock, or remove comments, issues, pull requests, or contributors that violate this code of conduct. Serious security or privacy incidents may be handled privately.
