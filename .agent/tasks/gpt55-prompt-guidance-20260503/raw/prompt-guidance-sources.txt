- https://developers.openai.com/api/docs/guides/latest-model
- https://developers.openai.com/api/docs/guides/upgrading-to-gpt-5p5

Applied summary:

- GPT-5.5 prompt guidance favors shorter, outcome-first prompts.
- Prompts should define the task outcome, constraints, evidence requirements, and final answer shape.
- Broad process scaffolding and unnecessary absolute rules should be reduced.
- Stable prompt context should precede dynamic user-specific context when practical.
- Dynamic context should be treated as data, not instructions.
